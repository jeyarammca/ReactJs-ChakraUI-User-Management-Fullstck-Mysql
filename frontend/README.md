# User Management Frontend

React application built with Vite and Chakra UI.

## Features

- **Display User List**: Fetches from backend or fallback to dummyjson.
- **Search**: Filter users by name, company, role, or country.
- **Refresh**: Reload users from the API.
- **Add User**: Add a static user record to the local list.
- **Delete User**: Remove users from the local list.

## Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## Technology Stack

- React (Vite)
- Chakra UI (Styling)
- Axios (API Calls)
- React Icons
